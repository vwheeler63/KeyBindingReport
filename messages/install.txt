KeyBindingReport is a Sublime Text Package that produces a wide variety
of reports about the current state of Sublime Text key bindings on the
system it is running on, with a choice of output formats.


Quick Start
-----------

Review the README to learn how to use it (Usage and Settings sections)
and enjoy!


Problems?
---------

If you encounter any problems with KeyBindingReport, or think that it
can be improved to make your work easier, please log an issue with
the KeyBindingReport issue tracker:

    https://github.com/vwheeler63/KeyBindingReport/issues
